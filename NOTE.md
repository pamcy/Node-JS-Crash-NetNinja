**SQL**
- Table
- Row
- Column
- Example: MySQL

**NoSQL**
- Collection
    - 類似 SQL 的 Table
    - user collection 儲存 user document
    - blog collectio 儲存 blog document
- Document
    - blog document 儲存 每一筆 blog 資料
    - 格式長的很像 JS 的 object or JSON
    - {
        "_id": ObjectId(12345),
        "title": "what a day!",
        "snippet": 'all about the...',
        "body": "blah blah blah..." 
    }
- Example: mangoDB
